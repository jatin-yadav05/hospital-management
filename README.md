# Hospital Management System

A comprehensive hospital management system built with React, Firebase, and Tailwind CSS.

## Features

- User Authentication
- Appointment Management
- Doctor Profiles
- Medical Store
- Patient Portal
- Health Metrics Tracking
- Order Management

## Technologies Used

- React
- Firebase (Authentication & Firestore)
- Tailwind CSS
- Framer Motion
- React Router
- Heroicons
